# TixHub

TixHub is a modern ticket management platform built to simplify how teams handle issues and support requests. With a clean, intuitive interface, it allows users to create, track, and resolve tickets efficientlyâ€”all in one place.

---

## Features

- Create, edit, and delete tickets  
- Set ticket priority and status  
- Filter and search tickets  
- Responsive and user-friendly interface  
- Login/logout with session management  

---

## Technologies Used

- **Frontend:** vue js
- **Storage:** LocalStorage (for tickets & session)  
- **Version Control:** Git & GitHub
